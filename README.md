# Cloud Warehouse (single admin login, multi-device)

Это веб-приложение складского учета с серверным хранением данных.
Данные лежат в SQLite (`warehouse.db`) на сервере, поэтому при доступе с разных устройств вы видите один и тот же актуальный склад.

Авторизация упрощена:
- только вход (регистрация отключена)
- вход по имени пользователя и паролю администратора

## Локальный запуск

```bash
cd "/Users/glebzajcev/Documents/New project"
python3 server.py
```

Открыть в браузере:

- `http://localhost:8000`

Данные для входа по умолчанию:
- имя пользователя: `admin`
- пароль: `admin123`

## Доступ с других устройств в одной сети

1. Узнайте IP компьютера, где запущен сервер (например `192.168.1.20`).
2. На другом устройстве откройте:
   - `http://192.168.1.20:8000`
3. Убедитесь, что порт `8000` не блокируется файрволом.

## Облачное размещение (интернет-доступ)

### Вариант 1: Render + Persistent Disk (быстрый старт)

1. Загрузите проект в GitHub.
2. На Render создайте `Web Service` (Python).
3. Start command:

```bash
python3 server.py
```

4. Добавьте Persistent Disk и смонтируйте в `/opt/render/project/src` (или перенесите `DB_PATH` в смонтированный путь).
5. После деплоя получите URL вида `https://your-app.onrender.com`.

### Вариант 2: Railway/Fly.io/VM

Аналогично запускаете `python3 server.py` и сохраняете `warehouse.db` на постоянном диске.
